Virtual Reality To PLC

On Virtual Reality To PLC project, we were etablishing communication between PLC and VR. We created HMI within VR environment, input from VR will be send to PLC, then PLC will respond based on data received and state of PLC. We can also get data from PLC data_block to VR.

This communication can be established using C# S7.Net. Which is only compitable with Siemens PLC's from S7-1500 to lower versions S7-300.

We can use this application for testing our ladder logic within the VR environment before we can execute it on actual Plant. And we will also try reducing number of sensors on actual Plant and implemt them on VR environment to save cost, to avoid electrical noise(NB : Dont tell Prof Pius About This As Yet <''>).

VR WORKOUT GAME

As we are having lot of gamers, and most of them usally do not exercise, which is not good for their health. We have decided create game that will help gamers to execise while playing game.

Program will get head position of gamer. Cubes will be generated randomly butween head position and half head position, and on left and right position. Every missed cube 1 point will be reduced, and every cube that you hit points will increment by 1. And there is a timer and target point, where you have to hit certain number of cubes in x seconds to unlock next level.
